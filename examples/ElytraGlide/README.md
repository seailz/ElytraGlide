# ElytraGlide
This is an example implementation of Flaps using the default "roll" effect to simulate the Elytra glide roll effect from Legacy console editions of Minecraft. Feel free to modify and expand upon this to create your own effects!

See [this video](https://www.youtube.com/watch?v=4TVhD3RRQrc) for reference.

This is the plugin only; you will also need the Flaps resource pack for this to work. Here's what it looks like in action:

There is some stuttering which is unfortunately unavoidable due to the limitations of creating smooth transitions with shaders (since they're stateless).
